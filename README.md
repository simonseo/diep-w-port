# DIEP-W Calculator

DIEP flap weight estimation calculator for breast reconstruction surgery.

## Repository Structure

- **`flutter/`** - Modern Flutter application (iOS, Android, Web, PWA) - **RECOMMENDED**
- **`cordova/`** - Legacy Cordova/Ionic web app (archived)

## Recommended Version

The **Flutter version** (`flutter/` directory) is the actively maintained version with:
- Modern Material Design 3 UI
- Dark mode support
- PWA capabilities
- Offline support
- Cross-platform (iOS, Android, Web)
- Calculation history with localStorage

See `flutter/README.md` for setup instructions.

## Live Demo

- **Flutter Web App**: [https://simonseo.github.io/diep-w-port/flutter/build/web/](https://simonseo.github.io/diep-w-port/flutter/build/web/)
- **Legacy Cordova App**: [https://simonseo.github.io/diep-w-port/cordova/](https://simonseo.github.io/diep-w-port/cordova/)

## About

This calculator implements the DIEP-W formula for estimating DIEP flap weight in breast reconstruction surgery, based on clinical research from Samsung Medical Center.
